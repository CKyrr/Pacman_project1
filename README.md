# Pacman_project1
https://inst.eecs.berkeley.edu/~cs188/archive/fa25/projects/proj1/

This repository contains my implementations for the widely recognized UC Berkeley CS188: Introduction to Artificial Intelligence Pacman projects. These projects focus on applying foundational AI algorithms to navigate an agent (Pacman) through complex maze environments, collect food, and efficiently evade adversarial ghosts.

# Project 1 — Search Algorithms:
Pac-Man finds optimal paths through maze environments using state-space search. Implemented algorithms:
- Depth-First Search (DFS)
- Breadth-First Search (BFS)
- Uniform Cost Search (UCS)
- A* Search with custom heuristics

These algorithms were applied to problems of increasing complexity: 
reaching a goal, collecting all food dots, and solving Traveling Salesman-style scenarios.

**Key concepts:** graph search, state representation, heuristic design, problem formulation.
